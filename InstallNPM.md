# How to Install NPM

## Preparation

```
sudo apt update
sudo apt install curl
```

## Install NVM

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
source ~/.bashrc
```

Check installation succeeded.
```
command -v nvm
```

##  Install NPM

```
nvm install v20.10.0
```
